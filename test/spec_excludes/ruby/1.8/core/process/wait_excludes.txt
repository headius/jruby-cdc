Process.wait returns its childs pid
Process.wait sets $? to a Process::Status
Process.wait waits for any child process if no pid is given
Process.wait waits for a specific child if a pid is given
Process.wait waits for a child whose process group ID is that of the calling process
Process.wait doesn't block if no child is available when WNOHANG is used
Process.wait always accepts flags=0
